# **App Name**: Aeon CRM Dashboard

## Core Features:

- User Authentication: Implements Firebase Authentication for user login, with consideration of storing user preferences and permissions. Accounts for a 'Welcome Back, Marci' scenario.
- Dashboard Metrics: Presents key metrics (Total Users, Total Products, Refunded Amount) in prominently displayed cards on the dashboard.
- Order List Component: Displays an interactive order list with features like sorting and pagination.
- Data Visualization: Includes charts illustrating monthly sales trends and the distribution between online and offline orders using D3.js.
- Predictive Analytics: A tool to analyzes sales data, including seasonality and regional trends, to predict future order volume and potential revenue, providing insights to inform marketing and inventory decisions.

## Style Guidelines:

- Primary color: Light grayish blue (#B0C4DE). A subdued and professional blue to create a calm and trustworthy environment for data presentation.
- Background color: Very light grayish blue (#F0F8FF). Provides a soft and clean backdrop to ensure that data and UI elements are clearly visible.
- Accent color: Muted cyan (#70DB93). Used sparingly to highlight interactive elements and key data points, without overwhelming the user.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its clean and modern appearance suitable for data-heavy interfaces.
- Use clean, simple icons that are easily understood to represent different functions and data categories.
- Follows a clear visual hierarchy, emphasizing key metrics at the top, interactive components in the middle, and analytical components at the bottom.
